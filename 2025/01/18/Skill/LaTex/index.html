LaTeX基本公式

靖2025-01-182025-01-18

LaTeX常用数学公式速查表

基础符号

  1. 上标和下标

    • 上标：x^2 →
    • 下标：x_i →
    • 组合：x^2_i →

  2. 分数

    • 简单分数：\frac{a}{b} →
    • 连分数：\frac{1}{\frac{1}{2}} →

  3. 根号

    • 平方根：\sqrt{x} →
    • n次方根：\sqrt[n]{x} →

运算符号

  1. 基本运算

    • 加：+ →
    • 减：- →
    • 乘：\times →
    • 除：\div →
    • 点乘：\cdot →

  2. 关系运算

    • 等于：= →
    • 不等于：\neq →
    • 大于：> →
    • 小于：< →
    • 大于等于：\geq →
    • 小于等于：\leq →
    • 约等于：\approx →

集合符号

  1. 基本集合
    • 属于：\in →
    • 不属于：\notin →
    • 包含：\subset →
    • 真包含：\subseteq →
    • 并集：\cup →
    • 交集：\cap →
    • 空集：\emptyset →

函数符号

  1. 三角函数

    • 正弦：\sin x →
    • 余弦：\cos x →
    • 正切：\tan x →
    • 余切：\cot x →

  2. 对数函数

    • 对数：\log x →
    • 自然对数：\ln x →
    • 以n为底：\log_n x →

极限与积分

  1. 极限

    • \lim_{x \to \infty} →
    • \lim_{x \to 0} →

  2. 积分

    • 定积分：\int_a^b f(x)dx →
    • 不定积分：\int f(x)dx →
    • 二重积分：\iint →
    • 三重积分：\iiint →

求和与连乘

  1. 求和

    • \sum_{i=1}^n →
    • \sum\limits_{i=1}^n →

  2. 连乘

    • \prod_{i=1}^n →
    • \prod\limits_{i=1}^n →

矩阵

  1. 基本矩阵

  1. 带括号的矩阵

希腊字母

  1. 常用小写

    • α：\alpha →
    • β：\beta →
    • γ：\gamma →
    • θ：\theta →
    • π：\pi →

  2. 常用大写

    • Γ：\Gamma →
    • Δ：\Delta →
    • Θ：\Theta →
    • Π：\Pi →

特殊符号

  1. 箭头

    • 右箭头：\rightarrow →
    • 左箭头：\leftarrow →
    • 双向箭头：\leftrightarrow →
    • 长右箭头：\longrightarrow →

  2. 数学符号

    • 无穷：\infty →
    • 偏导：\partial →
    • 点导数：\dot{x} →
    • 二阶导数：\ddot{x} →

常用公式示例

  1. 二次方程

    • ax^2 + bx + c = 0 →
    • 求根公式：x = \frac{-b \pm \sqrt{b^2-4ac}}{2a} →

  2. 泰勒展开

    • f(x) = f(a) + f'(a)(x-a) + \frac{f''(a)}{2!}(x-a)^2 + \cdots →

使用技巧

  1. 行内公式使用单个$符号包裹
  2. 独立公式使用双$$符号包裹
  3. 多行公式可以使用align环境
  4. 公式编号可以使用\tag{}命令

注意事项

  1. 在Markdown中使用LaTeX公式时，需要确保编辑器或平台支持LaTeX渲染
  2. 某些特殊符号可能需要转义
  3. 复杂公式建议使用独立公式模式，以获得更好的显示效果
  4. 注意公式中的空格和换行，它们可能影响渲染效果
